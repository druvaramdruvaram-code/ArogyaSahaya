# ArogyaSahaya

ArogyaSahaya is an Android healthcare assistance application developed to help users quickly access emergency services, nearby hospitals, ambulance support, and health-related assistance.

## Features

- Emergency SOS Support
- Ambulance Contact System
- Nearby Hospital Assistance
- Health Awareness Information
- User-Friendly Interface
- Android Native Application

## Tech Stack

- Android Studio
- Kotlin
- XML
- Firebase (if used)
- Google Maps API (if used)

## Installation

1. Clone the repository

```bash
git clone https://github.com/druvaramdruvaram-code/ArogyaSahaya.git
```

2. Open the project in Android Studio

3. Sync Gradle files

4. Run the app on Emulator or Android Device

## Folder Structure

- `app/` → Main application source code
- `res/` → UI resources
- `java/` or `kotlin/` → Application logic
- `AndroidManifest.xml` → App configuration
